1.1 音程

音乐界将两个音之间的距离称为音程（Interval），沿用历史习惯，以"性质+度数"来描述。

音程的基本概念

  • 根音（Bottom note）或下方音（Lower note）：音程中较低的音
  • 冠音（Top note）或上方音（Higher note）：音程中较高的音
  • 度数（Number）：根音到冠音之间跨过的字母数

度数示例：

  • C → C（相同音高）：只跨过C一个字母，称为一度（Unison）
  • C → 高八度C：跨过C-D-E-F-G-A-B-C共八个字母，所以是八度（Octave，8ve）
  • C → G：有五个字母，因此就是五度（Fifth，5th）

音程的性质

有一些音程虽然度数相同，但实际距离不同，例如C-F与C-F♭都是四度，但前者比后者多一个半音。为了区分，人们引入了音程的性质（Quality）：

  • 大（Major）：如大三度 C-E（Major 3rd，M3）
  • 小（Minor）：如小三度 C-E♭（Minor 3rd，m3）
  • 纯（Perfect）：如纯五度 C-G
  • 增（Augmented）：如增二度 C-D♯（Augmented 2nd，A2）
  • 减（Diminished）：如减四度 C-F♭

二度音程示例：

C到D♭、D、D♯都是二度，然而：

  • C-D♭只有一个半音：小二度（Minor 2nd，m2）
  • C-D有两个半音：大二度（Major 2nd，M2）
  • C-D♯有三个半音：增二度（Augmented 2nd，A2）

性质均不相同，分别叫做小二度、大二度、增二度。

1.2 和弦

人类最初的音乐是单声部的旋律，并没有出现和声。后来人们在一个声部之外，加入了其他声部与之配合，就产生了多声部音乐。

和声的起源

在实践中逐渐发现，某些音同时发声会产生协和悦耳的效果，而另一些则听起来不协和，于是就有了和声（Harmony）概念的雏形。

和弦的定义

音的各种搭配组合慢慢形成了一些固定的模式，就是今天所说的和弦（Chord）。和弦不仅有协和的，也有不协和的，只要用得恰当，都能极大地提高音乐表现力。

和声学

人们仔细研究和弦的结构、音响效果与相互关系，不断总结规律，最终建立了音乐的一门核心理论——和声学（Harmonics）。它主要解决两个问题："用什么和弦"与"怎么用和弦"。

我们用一个半音循环图来表示，在图中将三个音连起来，就形成了一个三角形。

这种"大三度+小三度"的结构就叫做大三和弦（Major triad）。

以任何一个音为根音，都能建立大三和弦，因此不考虑等音的情况下一共有12个。

大三和弦结构

"大三度+小三度"的结构就叫做大三和弦（Major triad），其中三个音都有各自的名称：

  • 根音（Root note）：C音，三度叠置的基础
  • 三音（3rd note）：E，根音之上三度
  • 五音（5th note）：G，比E高一个小三度，到根音C就是大三度+小三度=纯五度

和弦记号

人们习惯用根音的音名来代表整个大三和弦：

  • C-E-G和弦记作：C、Cmaj、CΔ
  • 这是现代产生的字母标记（Lead sheet symbol）或流行标记（Pop symbol）
  • 大量用于流行与爵士音乐

原位与转位

  • 原位（Root position）：根音放在最底下
  • 转位（Inverted position, Inversion）：其他和弦音放在最底下
  • 转位记号：C/E、C/G 分别表示把三音E、五音G放在最下面
  • 注意：这种基于音名的和弦记号是现代产生的

1.4 小三和弦

观察上面的图，在两个大三和弦中间是不是还有一个三角形呢？

小三和弦结构

E-G-B构成的三角形与大三和弦相比上下颠倒，属于另一种三和弦——小三和弦（Minor triad）

  • 结构："小三度+大三度"
  • 根音到三音为小三度
  • 三音到五音为大三度
  • 根音到五音仍为纯五度

音响和谐但情绪比较忧郁暗淡。

小三和弦记号

常用记号是在根音后面加上小写的m：

重要概念

  • 相同根音上的大、小三和弦只有三音不同，相差一个半音
  • 把大三和弦的三音降低半音，就变成了小三和弦；把小三和弦的三音升高半音，就变成了大三和弦，在图中表现为三角形的上下翻转
  • 少了三音，和弦就不确定是大三还是小三了。因此，三音决定了整个和弦的性质，地位仅次于根音，一般情况下不能省略
  • 五音主要作用是让音响更丰满，即使去掉也不改变和弦性质，时常可以省略
  • 省略五音时，和弦记号可保持不变，或加上"no5"、"omit5"表示

和声中的半音现象

三音的升降也揭示了和声的一个关键现象：半音的音程距离看似很近，但在和声中关系却很远，能显著改变音响效果。正因为如此：

  • 演唱、演奏、创作时发生失误，仅仅偏差半音，听起来却非常明显，非专业的听众也能轻易觉察
  • 利用这一点，有意使用半音变化，是和声的重要技法
