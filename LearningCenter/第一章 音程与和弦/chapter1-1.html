1.1 音程

音乐界将两个音之间的距离称为音程（Interval），沿用历史习惯，以"性质+度数"来描述。

音程中较低的音称为根音（Bottom note）或下方音（Lower note），较高的音称为冠音（Top note）或上方音（Higher note）。度数（Number）简单来说就是根音到冠音之间跨过的字母数。例如从C到相同音高的C，只跨过C一个字母，称为一度（Unison），到高八度的C则跨过C-D-E-F-G-A-B-C共八个字母，所以是八度（Octave，8ve）；从C到G有五个字母，因此就是五度（Fifth，5th）。

度数示例：

  • C → C（相同音高）：只跨过C一个字母，称为一度（Unison）
  • C → 高八度C：跨过C-D-E-F-G-A-B-C共八个字母，所以是八度（Octave，8ve）
  • C → G：有五个字母，因此就是五度（Fifth，5th）

音程的性质

有一些音程虽然度数相同，但实际距离不同，例如C-F与C-F♭都是四度，但前者比后者多一个半音。为了区分，人们引入了音程的性质（Quality）：

  • 大（Major）：如大三度 C-E（Major 3rd，M3）
  • 小（Minor）：如小三度 C-E♭（Minor 3rd，m3）
  • 纯（Perfect）：如纯五度 C-G
  • 增（Augmented）：如增二度 C-D♯（Augmented 2nd，A2）
  • 减（Diminished）：如减四度 C-F♭

二度音程示例：

C到D♭、D、D♯都是二度，然而：

  • C-D♭只有一个半音：小二度（Minor 2nd，m2）
  • C-D有两个半音：大二度（Major 2nd，M2）
  • C-D♯有三个半音：增二度（Augmented 2nd，A2）

性质均不相同，分别叫做小二度、大二度、增二度。
